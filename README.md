# 使用例子
## 记录pt的rust库hash
```pt_commit_tool record --pi-pt-root-path ../../../../pi_demo/projects/pi_pt --pt-root-path ../../../../GaiaPT/ ```

## 通过pt_commit_hash.txt文件恢复pt的rust库版本
```pt_commit_tool restore --pi-pt-root-path ../../../../pi_demo/projects/pi_pt --pt-root-path ../../../../GaiaPT/```
